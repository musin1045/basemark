# BaseMark 사진 기준점 자동 추천 계약

## 목적

현장 사진 또는 카메라 프레임에서 창호/프레임 모서리처럼 기준점으로 쓰기 쉬운 후보를 자동으로 먼저 제안한다.

이 단계의 역할은 `자동 확정`이 아니라 `수동 클릭 부담 감소`다.

## 입력

- 현재 화면에 표시된 사진 또는 캡처 프레임
- 렌더링된 RGBA 이미지 데이터
- 화면 폭/높이

## 처리

1. RGBA를 휘도값으로 변환한다.
2. 수평/수직 변화가 동시에 큰 지점을 corner response로 계산한다.
3. 화면 전체의 수직 에지와 수평 에지 강도를 따로 집계한다.
4. 화면을 4분면으로 나눠 각 분면에서 가장 강한 corner 후보를 찾는다.
5. 강한 수직선과 상단 수평선을 조합해 `벽 좌측 코너 / 우측 코너` 후보를 만든다.
6. 후보를 `창호 좌상단 / 우상단 / 좌하단 / 우하단 / 벽 좌측 코너 / 벽 우측 코너` 프리셋으로 매핑한다.
7. 신뢰도가 너무 낮은 후보는 버린다.

## 출력

- `suggestionId`
- `anchorPresetKey`
- `anchorKind`
- `label`
- `shortLabel`
- `point`
- `family`
- `confidence`

## 현재 컷라인

- 로컬 휴리스틱만 사용
- 창호 프레임 코너형 기준점과 상단 벽 코너형 기준점 우선
- 이미지 전체 자동 해석 없음
- 도면과 자동 정합 없음
- 최종 채택은 사용자가 직접 선택
