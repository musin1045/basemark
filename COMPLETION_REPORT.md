## 📱 검증 흐름

### Step 0: 홈 (Home)
```
목표: 검증 시작 의사 확인
내용: 앱 설명, 작동 방식, 필요한 것
시간: 즉시
액션: [검증 시작] → Step 1
```

### Step 1: 기준 표시 (Set References)
```
목표: 사진의 구조 기준점 2개 이상 표시
내용: 사진 업로드, 기준점 프리셋, 캔버스 클릭
시간: 30-60초
액션: 기준점 2개 → [다음] → Step 2
```

### Step 2: 비교 실행 (Run Comparison)
```
목표: 도면과 사진을 자동 비교
내용: 엔진 호출, 스피너, 결과 요약
시간: 1-2초
액션: 비교 완료 → [결과 확인] → Step 3
```

### Step 3: 결과 검토 (Review Results)
```
목표: 발견된 차이 항목 검토
내용: 결과 목록, 탭 필터링, 상세 정보, 리뷰 액션
시간: 30-60초
액션: 검토 완료 → [검증 완료] → 완료
```

### Step 4: 완료 (Complete)
```
목표: 검증 완료 확인
내용: 성공 메시지, 통계, 다시 시작 버튼
시간: 즉시
액션: [다시 검증하기] → 홈으로
```

**전체 소요 시간**: 2-3분 ✅

## 🚀 통합 방법

### 빠른 통합 (5분)

```bash
# 1. 저장소 준비
git clone https://github.com/musin1045/basemark.git
cd basemark

# 2. 파일 복사
cp basemark_v1_index.html src/ui/index.html
cp basemark_v1_styles.css src/ui/styles.css
cp basemark_v1_app_integrated.js src/ui/app.js

# 3. 서버 시작
npm install
npm run app

# 4. 브라우저 접속
open http://localhost:3000
```
