# 공수로 개인정보처리방침

시행일: 2026-03-23

## 1. 수집 및 저장하는 정보

공수로는 사용자가 직접 입력한 다음 정보를 처리합니다.

- 현장명, 작업 내용, 공수, 단가, 금액, 메모, 정산 상태
- 백업 및 복원에 사용되는 파일 정보
- 저녁 알림 설정 정보

위 데이터는 기본적으로 사용자 기기 내부 저장소에만 저장됩니다.

공수로는 회원가입 정보, 연락처 목록, 위치 정보, 광고 식별자, 생체 정보와 같은 추가 개인정보를 수집하지 않습니다.

## 2. 이용 목적

수집 또는 저장된 정보는 다음 목적에만 사용됩니다.

- 일일 작업 기록 저장 및 조회
- 기간별 정산 계산과 정산서 생성
- 백업 파일 내보내기, 가져오기, 복원
- 사용자가 켠 저녁 알림 예약

## 3. 외부 전송 및 제3자 제공

공수로는 입력 데이터를 운영 서버로 전송하지 않습니다.

다만 아래 기능은 사용자가 직접 실행한 경우에만 선택한 위치나 앱으로 정보가 전달될 수 있습니다.

- 백업 파일 저장 또는 공유
- 백업 파일 가져오기
- 정산서 PDF 공유

공수로는 이용자 정보를 판매하지 않으며, 광고 목적 또는 마케팅 목적으로 제3자에게 제공하지 않습니다.

## 4. 알림 기능

저녁 알림 기능을 켜면 알림 시간 설정과 로컬 예약 정보만 기기에 저장됩니다.

공수로는 원격 푸시 토큰을 수집하지 않으며, 서버를 통한 푸시 알림을 발송하지 않습니다.

## 5. 보관 및 삭제

입력 데이터는 사용자가 직접 삭제하거나 앱을 제거할 때까지 기기에 남을 수 있습니다.

사용자는 앱 내 기록 삭제, 백업 교체, 앱 삭제를 통해 데이터를 정리할 수 있습니다.
