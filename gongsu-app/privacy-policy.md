# 공수앱 개인정보처리방침

시행일: 2026-03-22

## 1. 수집하는 정보

공수앱은 사용자가 직접 입력한 현장명, 공수, 단가, 금액, 메모, 정산 상태, 백업 파일 정보를 기본적으로 기기 내부 저장소에 보관합니다.

## 2. 정보 이용 목적

수집된 정보는 다음 목적에만 사용됩니다.

- 일일 기록 저장과 달력 표시
- 정산 화면 계산 및 공유용 정산서 생성
- 백업 내보내기, 가져오기, 자동 복원
- 사용자가 활성화한 저녁 알림 예약

## 3. 외부 전송

앱은 기본적으로 입력 데이터를 외부 서버로 전송하지 않습니다.

다만 아래 경우에는 사용자가 직접 선택한 대상에게 정보가 전달될 수 있습니다.

- 백업 파일 공유
- 정산서 PDF 공유
- 사용자가 선택한 파일 가져오기

## 4. 알림

저녁 알림을 켜면 알림 시간 설정과 로컬 예약 정보가 기기에 저장됩니다. 이 기능은 원격 푸시 토큰을 수집하지 않으며, 서버 기반 알림 발송도 사용하지 않습니다.

## 5. 권한

앱은 다음 기능을 위해 권한을 사용할 수 있습니다.

- 알림 권한: 저녁 알림 표시
- 파일 접근 권한: 백업 가져오기 및 내보내기

## 6. 보관 및 삭제

입력 데이터는 사용자가 삭제하거나 앱을 제거할 때까지 기기에 남아 있을 수 있습니다. 앱 안의 삭제 기능, 백업 덮어쓰기, 앱 삭제를 통해 정리할 수 있습니다.
