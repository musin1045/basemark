Fixed Identity

BaseMark is a comparison engine first.

Structural anchor selection, normalized coordinates, local alignment, and missing/extra/position_diff candidate generation are the product core. Storage and reports are the operating layer.

Pipeline

  1. Select stable anchors for one bounded local segment.
  2. Define normalized checkpoints against those anchors.
  3. Align the local field segment and project expected positions.
  4. Generate candidate evidence for human review.

Field Workflow

  1. Use the camera or a captured frame as field evidence.
  2. Overlay anchors, expected points, and generated candidates.
  3. Review each candidate and confirm or reject in place.

Design Intent

The field screen is now the default working surface. Detailed scenario editing and raw JSON are available below in the advanced panel instead of taking over the whole page.
