한 줄 설명

BaseMark는 사진을 해석하는 AI가 아니라, 기준점을 잡아서 비교 후보를 계산하는 도구입니다.

먼저 현장의 변하지 않는 구조를 기준으로 잡고, 그 기준으로 예상 위치를 계산한 뒤, 사람이 확인할 후보를 보여줍니다.

앵커 0 기준으로 잡은 구조점
체크포인트 0 있어야 할 예상 위치
관측 요소 0 현장에서 실제 보이는 것
후보 0 마지막 실행 결과

앵커란?

앵커는 비교의 기준이 되는 구조점입니다. 예를 들면 문틀 모서리, 벽 코너, 창틀 끝점 같은 “잘 안 바뀌는 위치”입니다.

체크포인트란?

체크포인트는 “이쯤에 있어야 한다”는 예상 위치입니다. 절대 좌표가 아니라, 앵커 사이 비율로 정의됩니다.

관측 요소란?

관측 요소는 현장 사진에서 실제로 보이는 대상입니다. 스위치 박스, 콘센트 박스, 마감 포인트처럼 비교 대상이 되는 요소입니다.

후보란?

최종 판정이 아니라, 사람이 검토할 항목입니다. missing, extra, position_diff로 나뉩니다.

현장 비교

왼쪽에서 맞추고, 오른쪽에서 검토합니다.

1. 사진이나 카메라 화면 위에 앵커와 관측 요소를 맞춥니다. 2. 엔진이 후보를 계산합니다. 3. 오른쪽에서 후보를 확인하고 확정합니다.

사진 + 후보
